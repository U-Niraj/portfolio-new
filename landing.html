Web Development

Web development is the art of creating websites and web applications.
It involves designing the layout, adding functionality with code, and making sure everything works smoothly.
It's about bringing ideas to life on the internet for people to interact with.

How we Started

At FabLab, our journey into web development began with installing VS Code, a powerful tool for coding. We started by learning HTML, learning to structure web content. Next came CSS, adding style and design to our pages. Finally, we delved into JavaScript, bringing interactivity and functionality to our creations. It's been an exciting adventure of building and learning step by step.

Day 1

On day 1 at FabLab, we dived into HTML, the backbone of web pages. HTML, short for HyperText Markup Language, lets us structure content using tags. We began by creating headings, paragraphs, and lists, laying the foundation for building interactive and visually appealing websites.

  • Day 1 project

Day 2

On day two at FabLab, we ventured into CSS, the styling language that brings beauty to web pages. CSS, or Cascading Style Sheets, allows us to add colors, fonts, layouts, and animations to our HTML structure. We learned to make our websites visually stunning and user-friendly, enhancing the overall user experience.

  • Day 2 project

Day 3

On day three at FabLab, we delved deeper into CSS, learning advanced techniques and resolving doubts. With guidance from instructors, we tackled complex concepts like CSS frameworks, enhancing our ability to style web elements and create visually appealing websites. It was a day of mastering CSS intricacies and building confidence in our skills.

  • day 3 project

Day 4

On day four at FabLab, we delved into JavaScript (JS), a powerful scripting language for web development. JS enables us to add interactivity and functionality to our websites, from dynamic content creation to user input handling. We started by grasping variables and functions, setting the stage for crafting engaging and interactive web experiences.

  • Day 4 project

Day 5

On day five at FabLab, we ventured into Git and GitHub, essential tools for version control and collaboration in web development. We used Git for tracking changes, creating branches, and merging code. With GitHub, we hosted our website, enabling collaboration and showcasing our projects to the world. It was a pivotal day in mastering essential tools for efficient web development workflows.
Note : Day 5 project is the Day 4 project hosted online on Github

  • Day 5 project

Day 6

After learning HTML, CSS, JavaScript, and using Git and GitHub at FabLab, we started creating our own portfolio website. We structured the layout with HTML, added style and animations using CSS, and implemented interactive elements with JavaScript. Using Git and GitHub, we managed versions and deployed our website, showcasing our skills and projects to potential clients and collaborators.

  • day 6 project

What we learned

During the web development module at FabLab, we acquired essential skills in HTML, CSS, JavaScript, Git, and GitHub. HTML taught us content structuring, CSS added styling finesse, and JavaScript brought interactivity. Git and GitHub enabled version control and collaboration. Applying these skills, we crafted our portfolio website, showcasing our prowess in creating dynamic and engaging web experiences.
